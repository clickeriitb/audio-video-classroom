NPM is package manager for node.js.
Set registry command:
npm config set registry  https://registry.npmjs.org/

Note: npm config set registry  http://registry.npmjs.org/ for new version

set proxy command:
npm config set proxy http://username:password@proxy-server:port
npm config set https-proxy http://username:password@proxy-server:port

Installing Modules:
1.socket.io modules

npm install socket.io

2.serialport modules

npm install serialport   (works fine only for versions v0.8.14 or below)

To check if both the modules are installed correctly run the following command.
npm ls
